Descrizione dettagliata

Il server che gestisce le interazioni con i client.

Viene demandata a WordQuizzleServerService la vera gestione delle richieste. Non gestisce le sfide che invece vengono gestite da un thread apposito. Ogni connessione e' rappresentata da una Session contenente tutti i dati necessari per la corretta gestione delle richieste.

il server utilizza un selector
