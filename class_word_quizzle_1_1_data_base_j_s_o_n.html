Descrizione dettagliata

Database che gestisce gli utenti tramite un file JSON.

Implementa i comportamenti definiti dall'interfaccia DataBase attraverso un file JSON. Dopo una prima lettura si crea una copia in memoria su cui lavorare senza dover ri-leggere il file ogni volta. Fa il commit nel file dopo ogni operazione che modifica i dati. Il database e' rappresentato mediante una HashMap che associa ad un nome utente la sua istanza User. Implementazione Thread safe grazie alla gestione delle lock In lettura e scrittura.
